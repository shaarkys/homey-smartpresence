La aplicación funciona detectando puertos TCP cerrados de los smartphones en tu red Wi-Fi. Esto funcionará sin instalar la aplicación Homey y también te permitirá detectar invitados.

Instalación:

Para agregar un dispositivo, sigue estos pasos:

1. Revisa la dirección IP del smartphone. Esto se puede hacer desde la interfaz de administración del router Wi-Fi.
2. Haz una reserva estática para la dirección IP. También se hace desde la interfaz de administración del router Wi-Fi.
3. Añade el dispositivo ingresando un nombre y la dirección IP del smartphone. Marca las casillas 'Es invitado' o 'Es niño' para invitados o niños.

Para más información, haz clic en el enlace 'Visitar foro' en esta página.

Asegúrate de utilizar el MAC del TELÉFONO en la configuración de tu conexión Wi-Fi en tu teléfono y no un MAC ALEATORIO.
