L'application fonctionne en détectant les ports TCP fermés des smartphones sur votre réseau Wi-Fi. Cela fonctionne sans installer l'application Homey, et vous permettra également de détecter les invités.

Installation :

Pour ajouter un appareil, veuillez suivre ces étapes :

1. Vérifiez l'adresse IP du smartphone. Cela peut être fait depuis l'interface d'administration du routeur Wi-Fi.
2. Faites une réservation statique pour l'adresse IP. Également fait depuis l'interface d'administration du routeur Wi-Fi.
3. Ajoutez l'appareil en entrant un nom et l'adresse IP du smartphone. Cochez les cases 'Est invité' ou 'Est enfant' pour les invités ou les enfants.

Pour plus d'informations, cliquez sur le lien 'Visiter le forum' sur cette page.

Assurez-vous d'utiliser le MAC du téléphone dans les paramètres de votre connexion Wi-Fi sur votre téléphone et non un MAC aléatoire.
