앱은 Wi-Fi 네트워크에서 스마트폰의 닫힌 TCP 포트를 감지하여 작동합니다. Homey 앱을 설치하지 않고도 작동하며, 손님을 감지할 수도 있습니다.

설치:

장치를 추가하려면 다음 단계를 따르세요:

1. Wi-Fi 라우터의 관리자 UI에서 스마트폰의 IP 주소를 확인하세요.
2. IP 주소에 대해 정적 예약을 만드세요. 이것 또한 Wi-Fi 라우터의 관리자 UI에서 수행됩니다.
3. 이름과 스마트폰의 IP 주소를 입력하여 장치를 추가하세요. 손님 또는 아이의 경우 '손님 여부' 또는 '아이 여부' 체크박스를 선택하세요.

더 많은 정보를 보려면 이 페이지에서 '포럼 방문' 링크를 클릭하세요.

폰에서 Wifi 연결 설정 시 무작위 MAC이 아니라 PHONE MAC을 사용하고 있는지 확인하세요.
